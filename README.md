#### [管理缺位是管理上的大忌](other/README.md)


推荐理由： **管理上，最忌讳的事情是管理缺位,所谓管理缺位，是指某个组织或者某件事情处于没有管理状态而上级并未意识到。管理上不能有死角，这是管理的基本原则，管理缺位是管理上的大忌，一定会出现问题，只是时间早晚的问题而已**
